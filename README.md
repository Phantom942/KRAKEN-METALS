# KRAKEN METALS

Site vitrine pour KRAKEN METALS - Soudage de précision à Ivry-sur-Seine (94).

## Stack

- Next.js 14 (App Router)
- Tailwind CSS
- Lucide React
- TypeScript

## Démarrage

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Structure

```
src/
├── app/           # App Router (layout, pages, erreurs)
├── components/    # Composants React
└── lib/           # Constantes et utils
```
